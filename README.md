## Bookshelf Sinatra Project


# Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 

  Usage
  
  To use this app, fork and clone the repository. 
  
  Then run:
  
  $rake db:migrate 
  
  $rackup config.ru
